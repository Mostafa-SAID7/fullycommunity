# QA Controller Localization Keys

This document contains all the localization keys used in the QA Controller for both English (en) and Arabic (ar) languages.

## Success Messages

| Key | English (en) | Arabic (ar) |
|-----|-------------|-------------|
| `QA.Success.QuestionsRetrieved` | Questions retrieved successfully | تم استرجاع الأسئلة بنجاح |
| `QA.Success.QuestionRetrieved` | Question retrieved successfully | تم استرجاع السؤال بنجاح |
| `QA.Success.UserQuestionsRetrieved` | User questions retrieved successfully | تم استرجاع أسئلة المستخدم بنجاح |
| `QA.Success.RelatedQuestionsRetrieved` | Related questions retrieved successfully | تم استرجاع الأسئلة ذات الصلة بنجاح |
| `QA.Success.QuestionCreated` | Question created successfully | تم إنشاء السؤال بنجاح |
| `QA.Success.QuestionUpdated` | Question updated successfully | تم تحديث السؤال بنجاح |
| `QA.Success.QuestionClosed` | Question closed successfully | تم إغلاق السؤال بنجاح |
| `QA.Success.VoteRecorded` | Vote recorded successfully | تم تسجيل التصويت بنجاح |
| `QA.Success.QuestionBookmarked` | Question bookmarked successfully | تمت إضافة السؤال إلى المفضلة بنجاح |
| `QA.Success.QuestionUnbookmarked` | Question removed from bookmarks | تمت إزالة السؤال من المفضلة |
| `QA.Success.AnswersRetrieved` | Answers retrieved successfully | تم استرجاع الإجابات بنجاح |
| `QA.Success.AnswerCreated` | Answer created successfully | تم إنشاء الإجابة بنجاح |
| `QA.Success.AnswerUpdated` | Answer updated successfully | تم تحديث الإجابة بنجاح |
| `QA.Success.AnswerAccepted` | Answer accepted as solution | تم قبول الإجابة كحل |
| `QA.Success.CategoriesRetrieved` | Categories retrieved successfully | تم استرجاع الفئات بنجاح |
| `QA.Success.BookmarksRetrieved` | Bookmarks retrieved successfully | تم استرجاع المفضلة بنجاح |
